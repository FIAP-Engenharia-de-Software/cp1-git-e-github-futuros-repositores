## 📝 Sobre o Projeto
O **SocialConnect** é uma solução digital de vanguarda projetada para otimizar a conexão entre ecossistemas corporativos e seus usuários. Este repositório é o resultado do **CheckPoint 1 (CP1)**, focado em metodologias ágeis de versionamento e colaboração em equipe via Git/GitHub.

> **Objetivo:** Restaurar a integridade visual, corrigir inconsistências de dados e implementar novas camadas de interação na plataforma SocialConnect.

## ⚙️ Workflow & Governança de Código
Para garantir a qualidade e a conformidade com as exigências acadêmicas, implementamos um fluxo rigoroso:

* **🛡️ Branch Protection:** A branch `main` atua como ambiente de produção protegido. Nenhuma alteração direta é permitida.
* **🧩 Git Flow:** Divisão lógica em 8 frentes de trabalho (Feature, Fix, Docs).
* **📜 Conventional Commits:** 100% dos registros seguem o padrão semântico para garantir um histórico rastreável.
* **🤝 Code Review:** Todos os Pull Requests exigiram a aprovação de um par antes do merge final.

### Estrutura de Branches (Roadmap):
- `fix/imagens-e-cores` ➜ Recuperação de Assets e Identidade Visual.
- `fix/rodape-correcoes` ➜ Higienização de dados e correção ortográfica.
- `fix/sobre-texto-original` ➜ Restauração de conteúdo legado.
- `feature/sobre-equipe` ➜ Implementação de seção de talentos.
- `fix/menu-projetos` ➜ Reestruturação da arquitetura de navegação.
- `feature/secao-contato` ➜ Módulo de atendimento e suporte.
- `feature/redes-sociais` ➜ Integração com ecossistemas externos.
- `docs/readme-final` ➜ Documentação técnica e guia do projeto.
